Tutorial Penggunaan Aplikasi

Selamat datang di aplikasi peta interaktif!

  • Untuk menambahkan marker, klik pada lokasi di peta.
  • Data lokasi akan muncul di tabel di samping peta.
  • Gunakan tombol "Hapus" untuk menghapus marker dan data.
  • Klik tombol "Export to CSV" untuk mengunduh data lokasi.
  • Tabel akan berada di bawah peta saat menggunakan perangkat seluler.
